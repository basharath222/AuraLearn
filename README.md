# AuraLearn – Emotion-Aligned & Context-Aware AI Tutor

AuraLearn is an AI-powered learning assistant that adapts its teaching style based on students’ emotions and understanding level.
It extracts content from uploaded PDFs, generates simplified notes, quizzes, and tracks progress.

## Features
- Emotion detection via webcam
- AI-driven explanations and notes
- Personalized quiz generation
- Firebase login and progress tracking

## Tech Stack
Python • Streamlit • DeepFace • LangChain • Firebase • OpenCV • FAISS/Chroma

## Setup
1. Create virtual environment
2. Install dependencies (`pip install -r requirements.txt`)
3. Run app (`streamlit run app.py`)
